# Telegram Bot: Random Son Generatori

Bu bot `/start` komandasiga javoban 1 dan 10 gacha bo'lgan tasodifiy sonni chiqarib beradi.

## O'rnatish

1.  Bot tokenini olish uchun Telegramda [BotFather](https://t.me/BotFather) ga yozing va `/newbot` buyrug'ini kiriting.
2.  Olingan tokenni `index.js` faylidagi `YOUR_TELEGRAM_BOT_TOKEN` o'rniga qo'ying.
3.  Terminalda quyidagi buyruqni ishga tushiring:

```sh
npm install
npm start
```

## Ishlatish

Bot ishga tushgandan so'ng, unga `/start` yuboring va natijani ko'ring.
